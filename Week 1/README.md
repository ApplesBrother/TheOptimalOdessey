# Week 1 — Graphs, BFS & DFS

## 1. Introduction to Graphs
- What is a graph?
- Nodes and edges
- Cycles, Paths
- Directed vs Undirected graphs
- Weighted vs Unweighted graphs

## 2. Graph Representations
- Adjacency Matrix
- Adjacency List
- Space and Time Complexity

## 3. Breadth First Search, Depth First Search, BFS vs DFS
- Queue-based traversal
- Level-order exploration
- Shortest path in unweighted graphs
- Recursive DFS
- Iterative DFS using stack
- Traversal properties and applications
- Differences
- Complexity analysis
- When to use which approach

---

# Learning Objectives

By the end of this week, students should be able to:
- Understand graph structures
- Represent graphs efficiently
- Implement BFS and DFS
- Analyze traversal complexity
- Solve basic graph traversal problems
